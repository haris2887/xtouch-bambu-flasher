XTouch Bambu Bridge

Flash firmware to your CYD (ESP32-2432S028R) touchscreen

Requirements

  • Chrome or Edge browser (Web Serial API)
  • USB-A to Micro-USB cable (data capable, not charge-only)
  • XTouch CYD board (ESP32-2432S028R)

Instructions

  1. Connect your XTouch to this computer via USB
  2. Click Flash XTouch below
  3. Select the COM/Serial port when prompted
  4. Wait ~30 seconds for flashing to complete
  5. XTouch reboots → follow the on-screen WiFi setup

This tool flashes firmware via the browser using ESP Web Tools. No software install required.
To factory reset at any time: hold BOOT, press RST, release both, then re-flash here.
